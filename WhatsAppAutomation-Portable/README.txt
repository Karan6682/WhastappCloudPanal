==========================================
  WhatsApp Automation - Portable Edition
==========================================

This is a portable version of WhatsApp Automation.
No installation required!

HOW TO USE:
-----------
1. Double-click 'WhatsAppAutomation.bat' to start
2. Wait for the server to start
3. Open http://localhost:3001/app.html in your browser
4. Login with: admin / admin123

TO STOP:
--------
- Press Ctrl+C in the console window, or
- Run 'StopServer.bat'

DATABASE:
---------
This application connects to MSSQL database:
Server: 160.250.204.51
Database: pathosta_BulkWhatsAppsBusinessDB

REQUIREMENTS:
-------------
- Windows 7/8/10/11 (64-bit)
- Internet connection for WhatsApp API
- No Node.js installation required!

TROUBLESHOOTING:
----------------
- If port 3001 is busy, edit backend/config.js to change PORT
- Check Windows Firewall if connection fails
- Ensure MSSQL server is accessible
