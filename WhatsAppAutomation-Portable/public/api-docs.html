POST /webhook/send-message Send text message
Request Body (JSON)
{
  "phone": "918683916682",    // Required - with country code
  "message": "Hello from API!", // Required
  "session": "admin"           // Optional - default: admin
}
cURL Example
curl -X POST http://localhost:3001/webhook/send-message \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"phone":"918683916682","message":"Hello from API!"}'
Success Response
{
  "success": true,
  "message": "Message sent successfully",
  "phone": "918683916682",
  "timestamp": "2024-12-11T10:30:00.000Z"
}

                POST
                /webhook/send-pdf
                Send PDF document
            

            

                
Request (multipart/form-data)

                


phone: 918683916682     // Required
pdf: [file]             // Required - PDF file
caption: Your document  // Optional
session: admin          // Optional

                

                
                
cURL Example

                


curl -X POST http://localhost:3001/webhook/send-pdf \
  -H "X-API-Key: YOUR_API_KEY" \
  -F "phone=918683916682" \
  -F "pdf=@/path/to/document.pdf" \
  -F "caption=Here is your file"

                POST
                /webhook/send-bulk
                Send bulk messages
            

            

                
Request Body (JSON)

                


{
  "phones": ["918683916682", "919876543210"],
  "message": "Bulk message to all!",
  "session": "admin"
}

                

                

                     Messages are sent with 36-48 second delays to prevent WhatsApp blocking
                

            

        

        
        
        

            

                GET
                /webhook/status
                Check connection status
            

            

                
Query Parameters

                


session: admin  // Optional - default: admin

                

                
                
cURL Example

                


curl -X GET "http://localhost:3001/webhook/status?session=admin" \
  -H "X-API-Key: YOUR_API_KEY"

 Important Notes

            
    
                
  • All endpoints require X-API-Key header or api_key query parameter
    • 
                
  • Phone numbers must include country code (e.g., 91 for India)
    • 
                
  • WhatsApp must be connected before sending messages
    • 
                
  • Bulk messages include automatic delays to prevent blocking
    • 
                
  • PDF files must be sent as multipart/form-data
    • 
                
  • Maximum file size: 50MB
    • 
                
  • All webhook activity is logged with username "webhook"
