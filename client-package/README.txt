========================================
   WhatsApp Automation - Quick Start
========================================

INSTALLATION:
------------
1. Install Node.js from https://nodejs.org/ (LTS version)
2. Extract all files (keep folder structure intact)
3. Double-click QuickStart.bat

USAGE:
------
- Open browser: http://localhost:3001
- Login: admin / admin
- Connect WhatsApp by scanning QR code
- Send messages!

FILES INCLUDED:
--------------
- backend/        : Server files
- public/         : Web interface
- node_modules/   : Dependencies (pre-installed)
- QuickStart.bat  : Start server

NOTES:
------
- No internet required after first WhatsApp connection
- All data stored locally
- Change password from menu after first login

SUPPORT:
--------
If server doesn't start, make sure:
1. Node.js is installed
2. All files are extracted properly
3. Port 3001 is not in use
