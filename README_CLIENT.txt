============================================
   WHATSAPP AUTOMATION - OFFLINE SETUP
   Client Installation Guide
============================================

REQUIREMENTS:
   - Windows 10/11
   - Internet (sirf pehli baar setup ke liye)
   - Node.js (setup automatically bata dega)

FOLDER CONTENTS:
   - QuickStart.bat      -> App start karne ke liye
   - CLIENT_SETUP.bat    -> One-time setup (pehli baar run karo)
   - StopServer.bat      -> Server band karne ke liye
   - backend/            -> Server files
   - public/             -> Website files

============================================
   INSTALLATION STEPS
============================================

STEP 1: Node.js Install karo
   - https://nodejs.org/ pe jao
   - LTS version download karo
   - Install karo (Next Next karte raho)

STEP 2: Setup Run karo
   - CLIENT_SETUP.bat pe double-click karo
   - Wait karo jab tak complete na ho jaye
   - Desktop pe shortcut ban jayega

STEP 3: App Start karo
   - Desktop pe "WhatsApp Automation" shortcut pe click karo
   - Ya QuickStart.bat pe double-click karo

STEP 4: Login karo
   - Browser mein http://localhost:3001/app.html khulega
   - Username: admin
   - Password: admin123

STEP 5: WhatsApp Connect karo
   - "Connect WhatsApp" button dabao
   - QR code scan karo apne phone se
   - WhatsApp -> Settings -> Linked Devices -> Link a Device

DONE! Ab messages bhej sakte ho!

============================================
   DAILY USE
============================================

Roz app chalane ke liye:
1. Desktop pe "WhatsApp Automation" shortcut click karo
2. Browser khulega
3. Login karo aur use karo

Band karne ke liye:
- StopServer.bat click karo
- Ya terminal window band karo

============================================
   TROUBLESHOOTING
============================================

"Node.js not found" error:
   -> Node.js install karo: https://nodejs.org/

"Port already in use" error:
   -> StopServer.bat run karo pehle
   -> Phir QuickStart.bat run karo

QR code nahi aa raha:
   -> Page refresh karo (Ctrl+F5)
   -> "Connect WhatsApp" dobara click karo

Browser nahi khul raha:
   -> Manually jao: http://localhost:3001/app.html
